# Retail Sales Management System (Frontend)

## Overview
This is the React + Tailwind frontend for the Retail Sales Management System.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start development server:
   ```bash
   npm run dev
   ```
3. Build for production:
   ```bash
   npm run build
   ```

Make sure your backend server is running at `http://localhost:5000` for API connection.
